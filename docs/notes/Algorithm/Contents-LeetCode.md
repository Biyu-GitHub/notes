## 七、动态规划基础

**动态规划：**将原问题拆解成若干的子问题，同时保存子问题的答案使得每个子问题只求解一次，最终获得原问题的答案。

**最优子结构：**通过求子问题的最优解，可以获得原问题的最优解。
